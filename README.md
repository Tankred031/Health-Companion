# Health Companion

An interactive health and lifestyle web application focused on activity tracking, environmental awareness, hydration monitoring, and contextual wellness recommendations.

This project was created as my first fully independent interactive frontend application after completing foundational frontend development education.

The application combines:

* real-time activity tracking
* GPS movement monitoring
* weather integration
* hydration recommendations
* calorie calculations
* seasonal health logic
* personalized wellness systems

into a dynamic and responsive user experience.

---

## Technologies Used

* React
* JavaScript (ES6+)
* Vite
* Bulma CSS Framework
* Browser Geolocation API
* Open-Meteo API
* LocalStorage API
